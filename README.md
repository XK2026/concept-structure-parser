# concept-structure-parser

A Codex skill for learning new concepts by decomposing dense course material into explicit structures.

It guides Codex to parse:

- definition sentences as `A is B`
- action or process sentences as `A acts on B by doing C`
- qualifiers as scope, condition, boundary, feature, function, distinction, or example constraints

## Use

Place this folder under your Codex skills directory, then ask:

```text
Use $concept-structure-parser to parse these course materials.
```

The skill is designed for textbook excerpts, lecture slides, PDFs, notes, and university course concepts.
